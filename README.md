Este é o repositório do meu portfólio profissional, uma Single Page Application (SPA) moderna, interativa e responsiva construída para expor minha trajetória, habilidades e projetos de desenvolvimento.

## ✨ Funcionalidades Principais

- **Navegação Suave (Smooth Scroll):** Menu *sticky* com navegação in-page fluida.
- **Animações e Reveal (Framer Motion):** Elementos surgem e interagem conforme a rolagem da página.
- **Linha do Tempo Dinâmica:** Um histórico visual em zigue-zague com fotos (suporte a carrossel) e descrições acadêmicas e profissionais.
- **Detalhamento Avançado de Projetos:**
  - Sistema expansível de cards.
  - Carrossel fotográfico dedicado para interfaces e telas.
  - Abas informativas detalhando arquitetura e papéis no desenvolvimento (ex: estrutura multi-módulos do FWS).
- **Trilha Sonora Integrada (Audio Player):** Reprodutor flutuante e minimizável implementado do zero, com controle de volume, manipulação de faixas e extração inteligente de títulos locais.

## 🛠️ Tecnologias Utilizadas

- **Frontend Core:** React.js, Vite
- **Estilização & Grid:** Vanilla CSS (Design Glassmorphism), React-Bootstrap
- **Animações:** Framer Motion
- **Ícones:** React-Icons
- **Linguagens Base:** JavaScript, HTML5, CSS3

## 🚀 Como Executar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/NikolasLima29/Meu_portfolio_site.git
```

2. Acesse a pasta (caso tenha clonado com o nome padrão):
```bash
cd Meu_portfolio_site
```

3. Instale as dependências:
```bash
npm install
```

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

Acesse via navegador no endereço fornecido pelo terminal, geralmente `http://localhost:5173`.
